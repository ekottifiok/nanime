[nanime]{.text-primary} is a basic [Nuxt](https://nuxt.com) module designed to help you integrate [AnimeJS](https://animejs.com/)
into your Vue + Nuxt applications seamlessly.

## Why?

- SSR Safety
- Premade composables & components
- Zero Boilerplates
