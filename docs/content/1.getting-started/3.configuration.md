`nanime` is designed to work with zero configuration, but you can customize it to fit your needs.

## Module Options

You can pass options to the module directly in `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nanime'],
  nanime: {
    // Options
  }
})
```

### `composables`
- **Type**: `boolean`
- **Default**: `true`

Whether to auto-import composables (`useAnimate`, `useWaapiAnimate`).

```ts
nanime: {
  composables: false // Disable auto-imports
}
```

### `components`
- **Type**: `boolean`
- **Default**: `true`

Whether to register and auto-import components.

```ts
nanime: {
  components: false // Disable auto-imports of components
}
```

### `prefix`
- **Type**: `string`
- **Default**: `'A'`

Prefix for all registered components. See here for the list of components available

```ts
nanime: {
  prefix: 'Anime' // Adds the 'Anime' prefix to components.
}
```
