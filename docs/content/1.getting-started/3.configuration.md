# Configuration

`nanime` is designed to work with zero configuration, but you can customize it to fit your needs.

## Module Options

You can pass options to the module directly in `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nanime'],
  nanime: {
    // Options
  }
})
```

### `composables`
- **Type**: `boolean`
- **Default**: `true`

Whether to auto-import composables (`useAnimate`, `useWaapiAnimate`).

```ts
nanime: {
  composables: false // Disable auto-imports
}
```

### `components`
- **Type**: `boolean`
- **Default**: `true`

Whether to register transition components.

```ts
nanime: {
  components: false // Disable components
}
```

### `prefix`
- **Type**: `string`
- **Default**: `'A'`

Prefix for the registered components. For example, if prefix is `'A'`, components will be `<ASlide>`, `<AFade>`, etc.

```ts
nanime: {
  prefix: 'Anime' // Components become <AnimeSlide>, <AnimeFade>, etc.
}
```
