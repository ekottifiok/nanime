The `useAnimate` composable wraps the [animate](https://animejs.com/documentation/#animate) function from AnimeJS.

## Type Definition

```ts
type AnimeTargets = TargetsParam | MaybeElementRef | MaybeElementRef[]

function useAnimate(
  target: AnimeTargets,
  parameters: AnimeParams
): AnimeInstance
```

### Arguments

::caution
Passing an object created with `ref` will fail. You should pass a
`shallowReactive` or a `reactive` value as the target
::

- **target**: The value to animate. Can be a CSS selector, DOM element, Vue component, JS object, or an array of these.
- **parameters**: See [AnimeJS Animatable Properties](https://animejs.com/documentation/animation/animatable-properties).

## Usage

::code-group
```ts [script]
import { round } from '#nanime/utils';

const vector2D = shallowReactive({ x: 0, y: 0 });
useAnimate(vector2D, {
  x: 100,
  y: 150,
  duration: 10000,
  modifier: round(0),
  ease: "outElastic",
  alternate: true,
  loop: true,
});
```

```vue [template]
<template>
  <div class="text-center font-mono">
    {{ JSON.stringify(vector2D) }}
  </div>
</template>
```
::

## Return Value

Returns a `shallowReactive` object wrapping the AnimeJS instance.

::warning
**Do not destructure** the return value. The instance is wrapped in `shallowReactive` to maintain reactivity.
Destructuring will break the connection to the underlying AnimeJS instance updates.
::

```ts
// ❌ Avoid destructuring
const { play } = useAnimate('.target', { ... })

// ✅ Access methods directly
const animation = useAnimate('.target', { ... })
animation.play()
```

See [AnimeJS documentation](https://animejs.com/documentation/animation) for full list of parameters and properties.
