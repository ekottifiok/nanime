The `useWaapiAnimate` composable wraps the [waapi.animate](https://animejs.com/documentation/web-animation-api) function
from AnimeJS to help create [WAAPI](https://developer.mozilla.org/en-US/docs/Web/API/Web_Animations_API) powered animations.


## Type Definition

```ts
type WaapiElementTargets = VueInstance | HTMLElement | HTMLElement[] | SVGElement | SVGElement[] | NodeList | string | null
type WaapiTargets = MaybeRef<WaapiElementTargets> | MaybeRef<WaapiElementTargets>[]

function useWaapiAnimate(
  target: WaapiTargets,
  options: WAAPIAnimationParams
): WAAPIAnimation
```

### Arguments

- **target**: The DOM element(s) to animate.
- **options**: Object containing keyframes and animation options.

::warning
`useWaapiAnimate` **does not support** JS Objects as targets.
::

## Usage

::code-group
```ts [script]
import { stagger } from 'animejs/utils';

useWaapiAnimate('.square', {
  y: {
    to: [0, -15, 0],
    ease: 'out(4)',
    duration: 1000,
  },
  rotate: { from: -180, to: 0, ease: 'out(3)' },
  scale: { to: [.65, 1, .65], ease: 'inOut(3)' },
  duration: 500,
  delay: stagger(75),
  loop: true,
});

```

```html [template]
<div class="flex gap-2.5">
  <div 
    v-for="i in 6" :key="i"
    class="simple-box square" 
  />
</div>
```
::

::waapi-animate-demo

## Return Value

Returns a `shallowReactive` object wrapping the animation instance.

::warning
**Do not destructure** the return value to maintain reactivity.
::

```ts
const animation = useWaapiAnimate('.target', options)
animation.play()
```

See [AnimeJS WAAPI documentation](https://animejs.com/documentation/web-animation-api) for more details.
