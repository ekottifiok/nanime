Reactive wrapper for the [splitText](https://animejs.com/documentation/text) function from AnimeJS.

> A lightweight, responsive and accessible text utility function to split,
> clone and wrap lines, words and characters of an HTML Element.

## Type Definition

```ts
function useSplitText(
  target: SplitTextTargets,
  params: TextSplitterParams
): {
  lines: Ref<HTMLElement[]>,
  words: Ref<HTMLElement[]>,
  chars: Ref<HTMLElement[]>,
  properties: ComputedRef<NonFunctionProperties<TextSplitter>>,
  revert: () => void,
  split: () => void,
  refresh: () => void,
}
```

### Arguments

::field-group
  ::field{name="target" type="SplitTextTargets" required}
  The element or selector containing the text to split.
  ::

  ::field{name="params" type="TextSplitterParams"}
  See [AnimeJS Text documentation](https://animejs.com/documentation/text) for split options like `lines`, `words`, `chars`.
  ::
::

## Return Value

Returns an object with reactive references to the split elements and helper methods.

| Property | Type | Description |
| :--- | :--- | :--- |
| `lines` | `Ref<HTMLElement[]>`{lang="ts-type"} | Array containing line elements. |
| `words` | `Ref<HTMLElement[]>`{lang="ts-type"} | Array containing word elements. |
| `chars` | `Ref<HTMLElement[]>`{lang="ts-type"} | Array containing character elements. |
| `properties` | `ComputedRef<object>`{lang="ts-type"} | The underlying `TextSplitter` properties. |
| `split` | `() => void`{lang="ts-type"} | Manually trigger split. |
| `revert` | `() => void`{lang="ts-type"} | Revert the DOM to its original state. |
| `refresh` | `() => void`{lang="ts-type"} | Re-calculate splits (useful on resize). |

## API

### Types

```ts
type SplitTargets = HTMLElement | string
type SplitTextTargets = SplitTargets | MaybeElementRef<HTMLElement> | null | undefined | MaybeElementRef<VueInstance>
```

See [AnimeJS Text documentation](https://animejs.com/documentation/text) for more details.
