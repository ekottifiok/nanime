Reactive wrapper for the [splitText](https://animejs.com/documentation/text) function from AnimeJS.

> A lightweight, responsive and accessible text utility function to split,
> clone and wrap lines, words and characters of an HTML Element.

## Type Definition

```ts
type SplitTargets = HTMLElement | string
type SplitTextTargets = SplitTargets | MaybeElementRef<HTMLElement> | null | undefined | MaybeElementRef<VueInstance>

function useSplitText(
  target: SplitTextTargets,
  parameters: TextSplitterParams
): {
  lines: Ref<HTMLElement[]>,
  words: Ref<HTMLElement[]>,
  chars: Ref<HTMLElement[]>,
  properties: ComputedRef<NonFunctionProperties<TextSplitter>>,
  revert: () => void,
  split: () => void,
  refresh: () => void,
}
```

### Arguments

- **target**: The element or selector containing the text to split.
- **parameters**: See [AnimeJS Text documentation](https://animejs.com/documentation/text) for split options like `byLine`, `byWord`, `byChar`.

## Usage

::code-group
```ts [script]
const { chars, words } = useSplitText('.text', {
  chars: true,
  words: true
})

// Animate characters
useAnimate(chars, {
  y: [{ to: '-30%' }, { to: 0 }],
  delay: stagger(50),
  duration: 600,
  ease: 'inOutCirc',
  loop: true,
})
```

```vue [template]
<template>
  <span class="text">Hello World</h1>
</template>
```

## Return Value

Returns an object with reactive references to the split elements and helper methods.

- **lines**: `Ref` containing line elements.
- **words**: `Ref` containing word elements.
- **chars**: `Ref` containing character elements.
- **properties**: `ComputedRef` containing the underlying `TextSplitter` properties.
- **split()**: Manually trigger split.
- **revert()**: Revert the DOM to its original state.
- **refresh()**: Re-calculate splits (useful on resize).

See [AnimeJS Text documentation](https://animejs.com/documentation/text) for more details.
