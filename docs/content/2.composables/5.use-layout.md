The `useLayout` composable helps with layout animations.

## Type Definition

```ts
type AnimeLayoutTargets = DOMTargetSelector | MaybeElementRef<HTMLElement | VueInstance> | null | undefined

function useLayout(
  target: AnimeLayoutTargets,
  options: AutoLayoutParams
): {
  properties: ComputedRef<NonFunctionProperties<AutoLayout>>,
  record: () => void,
  revert: () => void,
  animate: (params?: LayoutAnimationParams) => void
}
```

### Arguments

- **target**: The wrapper element or selector for the layout transition.
- **options**: Auto layout parameters. See [AnimeJS Layout documentation](https://animejs.com/documentation/#layout).

## Usage

::code-group
```ts [script]
const items = ref([])
const { record, animate } = useLayout('.layout-wrapper', {
  duration: 500,
  ease: 'inOutQuad'
})

// Trigger layout animation after DOM changes
const addItem = () => {
  record()
  items.value.push({ name: items.value.length + 1 })
  nextTick(animate)
}
```

```vue [template]
<template>
  <div class="layout-wrapper">
    <div v-for="item in items" :key="item.id" class="layout-item">
      {{ item.name }}
    </div>
  </div>
</template>
```
::

## Return Value

Returns an object with helper methods to control the layout animation.

- **properties**: `ComputedRef` containing the current layout properties.
- **record()**: Record the current layout state.
- **revert()**: Revert the layout to its original state.
- **animate()**: Animate from the recorded state to the current state.

See [AnimeJS Layout documentation](https://animejs.com/documentation/#layout) for more details.
